Principal Components Analysis and Autoencoders: a dimensionality reduction comparison using a MNIST data set of American Sign Language

Project

Comparing PCA, (Denoising) Deep Autonencoders, and t-SNE with regards to dimensionality reduction task. As our domain, we take a MNIST data set of American Sign Language. We also study the clustring properties of the mentioned methods on the data set.

Domain

Study

Skills used

Python (Pytorch), Deep Learning (Denoising Deep Autoencoders), Machine Learning (k-means, K-medoids, ...), Econometrics (PCA, t-SNE)

The main contribution of this reserach project is the way how we managed to reproduce the pictures from the Americal Sign Language data set after reducing them from 784 dimensions (pixels) to 13.
