Aspect-Based Sentiment Quantification

Project

The idea of this thesis was to estimate the aggregated sentiment with regards to each of the product aspects across a selection of user reviews. Using Quantification methods, this can be done more efficiently then via simple Classification.

Domain

Study

Skills used

Python (Tensorflow), bi-LSTM with multi-layer attention, hierarchical attention

Abstract: In the current literature, many methods have been devised for sentiment quantification. In this work, we propose AspEntQuaNet, one of the first methods for aspect-based sentiment quantification. It adjusts the state-of-the-art QuaNet deep learning method for sentiment quantification by considering aspects and ternary sentiment classification instead of binary sentiment classification with respect to these aspects. Other sentiment quantification methods have also been adapted for ternary sentiment quantification instead of binary sentiment quantification. Using the adapted version of SemEval 2016 dataset for aspect-based sentiment quantification, we show that AspEntQuaNet is superior to all other considered methods on the basis of 4 various aspect categories. Furthermore, AspEntQuaNet is particularly good when evaluated by Kullback-Leibler divergence, oftentimes outperforming ordinary classification by more than 10 times.
